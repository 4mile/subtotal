Subtotal.js Examples

Subtotal.js is an open-source JavaScript plugin for PivotTable.js. It renders subtotals of rows and columns with the ability to expand and collapse rows. Its written by Nagarajan Chinnasamy at Mindtree Ltd..

PivotTable.js is an open-source Javascript Pivot Table implementation written by Nicolas Kruchten at Datacratic.
