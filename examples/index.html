Subtotal.js

Subtotal.js is an open-source JavaScript plugin for PivotTable.js. Subtotal.js renders rows and columns of a pivot table with subtotals and lets the user to expand and collapse them. Its originally written by Nagarajan Chinnasamy at Mindtree.

PivotTable.js is an open-source Javascript Pivot Table implementation written by Nicolas Kruchten at Datacratic.
